# Electro Browser Linux edition

- This is the linux version of electro browser.*Be warned: a release will be made faster for windows but a linux release will eventually come out.*

> This source code has been tested on: **Ubuntu/zorin os** and will probably work on all ubuntu/debian based devices

---

## Run The App

### you need:

* Python **3.10+ / Python 3.14 Recommended**
* `pip`

### now:
- Run "run.sh" *it will automatically download pyside6*
- **Warn: you'll need to keep .venv file for it to run!**

- If u somehow somehow broke .venv just run 
<pre>pip install pyside6</pre>
- then run main.py
